Matéria Descrição
Construção de Software para Web Esta matéria aborda os princípios e técnicas de desenvolvimento de aplicações web, incluindo front-end e back-end. Os alunos aprendem a criar, testar e implementar sites e aplicações, utilizando linguagens como HTML, CSS e JavaScript, além de frameworks e boas práticas de programação.
Design e Desenvolvimento de Banco de Dados I Foca na teoria e prática da modelagem, implementação e gerenciamento de bancos de dados. Os alunos aprendem sobre SQL, design de esquemas, normalização de dados e como criar bases de dados eficientes para suportar aplicações.
Experiência e Interface com o Usuário Esta disciplina explora o design de interfaces e a experiência do usuário (UX). Os alunos aprendem sobre usabilidade, design centrado no usuário, prototipagem e testes de interface, visando criar produtos digitais intuitivos e agradáveis.
Lógica para Computação Enfatiza os fundamentos da lógica aplicada à computação, incluindo raciocínio lógico, estruturas lógicas e algoritmos. Os alunos desenvolvem habilidades de resolução de problemas e aprendem a construir algoritmos eficientes para resolver desafios computacionais.
Fundamentos de Tecnologia da Computação Oferece uma visão geral das tecnologias subjacentes à computação, abrangendo hardware, software, sistemas operacionais e redes. Os alunos exploram como essas tecnologias interagem e sustentam as aplicações e sistemas computacionais modernos.

Serviços

Serviços de desenvolvimento de sites estáticos, dinâmicos e avançados.

Tipo de Serviço Descrição Preço
Página Estática (1-2 páginas) Criação de uma página web simples que utiliza HTML para estrutura e CSS para estilização. É ideal para sites que não requerem interação com o usuário ou atualizações dinâmicas, como portfolios ou páginas de contato. R$ 800 a R$ 1.500
Página Dinâmica (3-5 páginas) Criação de uma página que interage com o usuário e pode exibir conteúdo variado, geralmente utilizando tecnologias como JavaScript, PHP ou frameworks como React. Ideal para blogs, sistemas de gerenciamento de conteúdo, etc. R$ 2.000 a R$ 5.000
Página Avançada (+5 páginas) esenvolvimento de um site ou aplicação web com recursos avançados, como animações complexas, integração com APIs, banco de dados, autenticação de usuários, e responsividade total. Pode incluir aplicações como e-commerces, plataformas sociais ou sistemas customizados. R$ 8.000 a R$ 20.000
